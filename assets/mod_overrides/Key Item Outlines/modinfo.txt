Key Item Outlines
Created for the payday2-modding repository.

Description: Adds persistent outlines to key item world pickups
(keycards and meth ingredient containers) to make them easier to spot
during heists.

This mod works by overriding the material_config files of the world pickup
units with the CONTOUR render template flag, giving those models a permanent
glowing yellow outline — the same technique used by Gage Package Contours.

Install instructions:
This folder should be placed inside "...PAYDAY 2/assets/mod_overrides/".

Units covered (world pickup models, not carry-on-back models):
  gen_pku_keycard        – keycard pickup (keycards in various heists)
  gen_pku_toolbag        – small tool bag (muriatic acid / meth ingredient)
  gen_pku_toolbag_large  – large tool bag (caustic soda / meth ingredient)
  gen_pku_cage_bag       – cage bag (hydrogen chloride / meth ingredient)
